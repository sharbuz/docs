# Update Guides

Guides for updating AI/Run components to newer versions.

## Component Updates

- **[CodeMie Platform](./codemie/update-version)** - Update AI/Run CodeMie platform with automated or manual procedures
- **[Keycloak](./keycloak-upgrade/)** - Upgrade Keycloak identity and access management
- **[Elasticsearch & Kibana](./elasticsearch-kibana-upgrade)** - Upgrade observability stack components

## Before You Update

:::warning Pre-Update Checklist

- Back up all critical data and configurations
- Test updates in a non-production environment
- Plan for maintenance windows
  :::

:::tip
Each component guide includes troubleshooting steps for common issues. If you need additional help, contact your support team with detailed information about your environment and any errors encountered.
:::
