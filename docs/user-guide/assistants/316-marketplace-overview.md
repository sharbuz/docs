# 3.16 Marketplace Overview

The Marketplace is a centralized hub within the Assistants section where you can discover, explore, and deploy ready-to-use assistants created by the community. It provides streamlined access to all global assistants in one convenient location.

![Marketplace interface showing available assistants](../images/image172.png)

## Key Features

- Curated catalog of community-contributed assistants

- User engagement through likes/dislikes on assistant cards

- Adoption metrics showing how many unique users have adopted each solution

- Popular ranking with most adopted assistants appearing at the top

- Enhanced filtering that's always visible and allows easy combination of filters

- Personal assistant filter to find assistants you've created with a simple checkbox
