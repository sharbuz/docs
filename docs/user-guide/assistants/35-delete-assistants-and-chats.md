# 3.5 Delete Assistants and Chats

An excessive amount of redundant assistants affects navigation. The same case is related to chats. In this case, cleanup might be the way to go. In this section, we will learn how to delete both assistants and chats.

## Delete Assistants

To delete assistants, follow the steps below:

1. Navigate to **AI/Run CodeMie** → **Assistants** → **Project Assistants**

2. Click the **Actions** button and select **Delete**:

   ![Delete Assistant](../images/image188.png)

3. Confirm the assistant's deletion:

   ![Confirm Deletion](../images/image102.png)

:::note
As soon as you delete the assistant, the chats will not be operating any longer. You will have to delete the chats manually.
:::

## Delete Chats

To delete a chat with an assistant:

1. Click the **Chats** icon

2. Select the necessary conversation

3. Click the **Settings** button and select **Delete**:

   ![Delete Chat](../images/image52.png)

4. Confirm the chat deletion:

   ![Confirm Chat Deletion](../images/image19.png)
