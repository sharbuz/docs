# Git Integration Overview

Git integration enables your assistants to interact with version control systems, manage repositories, review code, create pull requests, and perform various Git operations.

## Available Git Integrations

AI/Run CodeMie supports Git integration with multiple platforms:

### [GitHub / GitLab / Bitbucket](./git-github-gitlab-bitbucket)

Integration with the most popular Git hosting platforms:

- **GitHub**: Public and private repositories, pull requests, issues, actions
- **GitLab**: Projects, merge requests, CI/CD pipelines, issues
- **Bitbucket**: Repositories, pull requests, pipelines

**Key capabilities:**

- Repository browsing and file operations
- Pull request and merge request management
- Branch operations and code review
- Issue tracking and project management
- CI/CD pipeline interaction

### [Azure DevOps](./git-azuredevops)

Comprehensive Azure DevOps integration including:

- **Azure Repos**: Git repository management
- **Azure Wiki**: Documentation and knowledge base
- **Azure Boards**: Work item tracking and management
- **Azure Test Plans**: Test case management and execution

**Key capabilities:**

- Repository and branch management
- Work item creation and updates
- Wiki page management
- Test plan and test case operations

---

## Integration Requirements

All Git integrations require:

- ✅ Integration configuration with credentials
- ✅ Appropriate access permissions
- ✅ Repository or project access rights

For detailed setup instructions, click on the specific Git platform above.
